## 아이디어 1: 한국 전통 골목길 감성 (Korean Alley Aesthetic)

**Design Movement:** 한국 전통 골목길과 현대적 미니멀리즘의 조화

**Core Principles:**
- 따뜻한 황토색과 크림색 기반의 자연스러운 색감
- 손글씨 느낌의 타이포그래피와 깔끔한 본문 폰트의 대비
- 골목길을 걷는 듯한 스크롤 경험

**Color Philosophy:** 
- Primary: 따뜻한 테라코타(#C4A77D) - 청주 골목의 흙담 느낌
- Secondary: 크림 화이트(#FDF8F3) - 한지의 부드러움
- Accent: 진한 갈색(#5C4033) - 오래된 목재의 깊이

**Layout Paradigm:** 
- 세로 스크롤 기반의 카드 레이아웃
- 각 맛집이 골목의 가게처럼 나열되는 구조
- 비대칭적인 카드 배치로 자연스러운 리듬감

**Signature Elements:**
- 손그림 스타일의 지도 마커
- 도보 시간을 발자국 아이콘으로 표현
- 한지 질감의 배경 텍스처

**Interaction Philosophy:** 
- 부드러운 페이드인 애니메이션
- 카드 호버 시 살짝 들어올려지는 효과
- 스크롤에 따른 패럴랙스 효과

**Animation:**
- 페이지 로드 시 위에서 아래로 순차적 등장
- 부드러운 이징 함수 사용 (ease-out)
- 지도 마커 펄스 애니메이션

**Typography System:**
- 제목: Noto Serif KR (세리프, 전통적 느낌)
- 본문: Pretendard (깔끔한 가독성)
</idea>

## 아이디어 2: 보드게임 테마 인터페이스 (Board Game Theme)

**Design Movement:** 보드게임 컴포넌트와 게이미피케이션 디자인

**Core Principles:**
- 보드게임 말(미플)을 활용한 시각적 아이덴티티
- 게임 보드 위를 이동하는 듯한 인터랙션
- 순위 시스템을 게임 점수처럼 표현

**Color Philosophy:**
- Primary: 딥 네이비(#1A365D) - 고급 보드게임 박스 느낌
- Secondary: 골드(#D69E2E) - 1등 트로피, 승리의 상징
- Accent: 따뜻한 오렌지(#ED8936) - 미플의 대표 색상
- Background: 크래프트 페이퍼(#F7FAFC) - 게임 보드 느낌

**Layout Paradigm:**
- 그리드 기반의 게임 보드 레이아웃
- 각 맛집이 게임 칸처럼 배치
- 중앙에 미플 카페를 두고 방사형으로 맛집 배치

**Signature Elements:**
- 미플 모양의 마커와 아이콘
- 주사위 굴리기 인터랙션 (랜덤 추천)
- 순위를 메달/트로피로 표현

**Interaction Philosophy:**
- 클릭 시 카드가 뒤집어지며 상세 정보 표시
- 미플이 이동하는 애니메이션
- 게임적 요소를 통한 재미 요소

**Animation:**
- 카드 플립 애니메이션
- 미플 바운스 효과
- 점수판 스타일의 숫자 롤링

**Typography System:**
- 제목: Fredoka One (둥글고 친근한 느낌)
- 본문: Noto Sans KR (깔끔한 가독성)
</idea>

## 아이디어 3: 모던 푸드 매거진 스타일 (Modern Food Magazine)

**Design Movement:** 에디토리얼 디자인과 푸드 포토그래피 중심

**Core Principles:**
- 대담한 타이포그래피와 여백의 미학
- 음식 사진이 주인공이 되는 레이아웃
- 잡지를 넘기는 듯한 세련된 경험

**Color Philosophy:**
- Primary: 다크 차콜(#1A1A2E) - 고급스러운 배경
- Secondary: 퓨어 화이트(#FFFFFF) - 깔끔한 대비
- Accent: 비비드 레드(#E53E3E) - 맛있는 음식의 열정
- Highlight: 골드(#F6E05E) - 프리미엄 강조

**Layout Paradigm:**
- 매거진 스프레드 스타일의 풀스크린 섹션
- 왼쪽 이미지 + 오른쪽 텍스트의 교차 배치
- 대담한 숫자(순위)가 배경으로 깔리는 레이아웃

**Signature Elements:**
- 큰 순위 숫자가 배경에 워터마크처럼 표시
- 도보 시간을 타임라인 형태로 시각화
- 에디터 추천 배지 스타일

**Interaction Philosophy:**
- 스크롤 스냅으로 섹션별 이동
- 이미지 호버 시 줌인 효과
- 부드러운 페이지 전환

**Animation:**
- 텍스트 마스크 애니메이션
- 이미지 패럴랙스 스크롤
- 순차적 요소 등장 (stagger)

**Typography System:**
- 제목: Playfair Display (클래식 에디토리얼)
- 본문: Noto Sans KR (가독성)
- 강조: Bebas Neue (대담한 숫자)
</idea>

## 선택된 디자인: 아이디어 2 - 보드게임 테마 인터페이스

**선택 이유:**
1. 미플 보드게임 카페의 정체성과 완벽하게 연결
2. 맛집 순위를 게임 점수처럼 표현하여 재미 요소 추가
3. 미플(Meeple) 아이콘을 활용한 독특한 시각적 아이덴티티
4. 사용자들이 보드게임 모임 후 자연스럽게 연결되는 경험

**구현 계획:**
- 히어로 섹션: 미플 카페를 중심으로 한 지도 스타일 인트로
- 맛집 리스트: 게임 보드 스타일의 카드 그리드
- 각 카드: 순위 메달 + 도보 시간 + 핵심 정보
- 상세 정보: 카드 클릭 시 모달로 표시
- 지도 통합: Google Maps로 실제 위치 표시
